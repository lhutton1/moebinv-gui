How to add a tooltip to an element

For this example we will create a tooltip for the 'create cycle' tool bar button. First we need to go to the MainWindows constructor. Then the button needs to be identified, the create cycle button in this case is named 'actionCreate_Cycle'. Then we add a tool tip to the button like so:

ui->actionCreate_Cycle->setToolTip("Create a cycle by using the currently selected relations.");

For all widgets the QWidget::setToolTip() function can be used to add a tool tip which becomes visible on hover.
