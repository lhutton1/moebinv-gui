How to create a new menu

To add a new menu to the menu bar, we first need to create a new menu. (If the menu is to be visible when the program is loaded this should be placed in the MainWindow constructor.)

// PLACE NEW MENU ITEMS HERE....
QMenu *newMenu = new QMenu("new menu text", nullptr);

// now add the relevent ations to the new menu
QAction *newAction = new QAction("new action text", newMenu);
newMenu->addAction(newAction);
...

Now we need to add the menu to the menu bar.

ui->menuBar->addMenu(newMenu);

To run a function when a menu item is clicked we need to connect the actions triggered signal to a slot.

connect(newAction, &QAction::triggered, this, &MainWindow::slot name);

For more information visit: http://doc.qt.io/qt-5/qtwidgets-mainwindows-menus-example.html
